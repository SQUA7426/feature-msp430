gesamt-Zeit := ACLK * 0.270 = 540_000 Ticks

problem: 2**16 => 65_535 

Da 2**16 nicht in Ticks => ACLK / bit-Teiler
  Neue Frequenz := 125 kHz

  => neue Anz benoetigter Ticks:  125_000 Hz * 0.270 = 33_750 Ticks

  => CCR0 => 33_750 - 1 = 33_749

    => bin(33_749) = 1000 0011 1101 0101    <= TA0CCR0
